# paranoid

> A single Android app hosting many small apps (mini-apps).

## Overview

Paranoid is an Android app that wraps multiple self-contained mini-apps in a WebView. Each mini-app is vanilla HTML/CSS/JS — no frameworks, no build step. The hub screen lists all available mini-apps and lets you launch them.

## Structure

- Mini-apps live at the repo root, each in its own folder with `index.html`, `style.css`, `script.js`
- Each mini-app has a `spec/functionality.md` describing its purpose and `sessions/` for dev notes
- The Android project lives in `android/` — a Kotlin WebView shell
- `build-metadata.sh` auto-discovers mini-apps and generates `apps-metadata.json`
- `justfile` provides scaffolding (`just new app-name`) and build commands

## Design

- Dark theme (#121212 background), mobile-first, touch-friendly
- Offline-capable — all content runs locally in WebView
- Vanilla JS/CSS only, no external dependencies

## Commands

- `just new <name>` — scaffold a new mini-app
- `just serve` — preview in browser
- `just build` — build Android APK
- `just install` — install on connected device
