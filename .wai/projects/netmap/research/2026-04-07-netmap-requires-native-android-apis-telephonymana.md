NetMap requires native Android APIs (TelephonyManager, FusedLocationProvider, foreground service, Room DB, MapLibre). It cannot run as a WebView mini-app. This is the first native Compose screen in paranoid, which means the app architecture needs to evolve: the hub stays WebView-based for simple apps, but native screens like NetMap are accessible via a JavaScript bridge or native navigation. Key dependencies: Hilt DI, Room, MapLibre Native, Accompanist Permissions, kotlinx.serialization.
